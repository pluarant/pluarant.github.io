Alternate Way!

We use the popular programming Q&A website Stack Overflow to field technical questions about the Mesibo API for Android, iOS and Web services. Stack Overflow is a collaboratively-edited question and answer site for programmers.

The site is not run by Mesibo, but you can sign in using your Mesibo account. It's a great place to ask technical questions about developing and maintaining your app. Members of the Mesibo api and Mesibo api APIs team monitor several Mesibo api and Mesibo api related tags on Stack Overflow. You can look for Mesibo api API for Android-specific topics by adding Mesibo-places-api+android. You can look for Mesibo api APIs topics by adding Mesibo-maps to your search query
